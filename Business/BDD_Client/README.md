# 🧩 Base Clients — Data Engineering & CRM

## 🎯 Objectif

Construire une base clients fiable et exploitable à partir de sources multiples.

---

## ⚙️ Pipeline de données

- Collecte multi-sources :
  - bornes
  - click & collect
  - commandes téléphoniques
  - jeux concours

- Automatisation avec Python :
  - Selenium
  - Pandas

---

## 🧹 Data cleaning

- Suppression des doublons
- Gestion des valeurs manquantes
- Uniformisation des formats

---

## 📈 Résultat

- Base clients étendue de 4 000 à 7 700 contacts (+92%)

---

## 🛠️ Outils

- Python (Pandas, Selenium)
